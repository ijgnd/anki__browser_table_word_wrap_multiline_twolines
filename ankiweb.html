Use this at your own risk. Uploaded so that other people can find it and maybe improve it. This add-on adds word-wrap to the Browser table. This add-on is intended as a companion to the add-ons Browser Maximize/Hide Table/Editor and Browser Card Info Bar. You can configure the shortcut in the add-on settings. # Limitations - To disable this add-on you need to close and reopen the browser. - This add-on slows down the browser. If someone has a fixes for these they would be very welcome. Thanks @/u/brunzus for adding a crucial line of code. For background info see this reddit thread. If you have code for improving this add-on let me know in the review section, via reddit (/u/ijgnord) or on github.
